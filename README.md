# SQL-Python_Ecommerce

## 📌 Project Overview

This project demonstrates SQL data analysis using Python on an E-commerce dataset. It covers SQL querying, data extraction, aggregation, joins, window functions, and data visualization to generate business insights.

The project was completed as a hands-on learning exercise to strengthen SQL and Python skills using a real-world dataset.
